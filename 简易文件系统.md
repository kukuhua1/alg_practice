# 简易文件系统

## 题目概述

设计并实现一个简易文件系统，支持以下操作：

- `FileSystem()`：初始化系统，当前目录为根目录 `/`
- `make_dir(dir_name)`：在当前目录下创建目录
- `create_file(file_name)`：在当前目录下创建文件
- `change_dir(path_name)`：切换当前目录，支持绝对路径和相对路径
- `remove(name)`：删除当前目录下的文件或目录
- `list_dir()`：返回当前目录下的内容，目录在前、文件在后，且各自按字典序升序

## 解题思路

### 1. 数据结构

用一个目录节点表示文件系统中的目录：

- `dirs`：当前目录下的子目录，类型为 `dict[str, Directory]`
- `files`：当前目录下的文件集合，类型为 `set[str]`

文件系统对象中维护两个指针：

- `root`：根目录
- `current`：当前所在目录

### 2. 操作实现

- `make_dir(dir_name)`
  - 如果名字和当前目录下已有文件或目录重复，返回 `False`
  - 否则创建新目录并返回 `True`

- `create_file(file_name)`
  - 如果名字和当前目录下已有文件或目录重复，返回 `False`
  - 否则创建文件并返回 `True`

- `change_dir(path_name)`
  - 若 `path_name == ""`，保持当前目录不变，返回 `True`
  - 若以 `/` 开头，从根目录开始解析
  - 否则从当前目录开始解析
  - 只允许进入目录，若路径中的某一段不存在，则返回 `False`

- `remove(name)`
  - 若是文件，直接删除
  - 若是目录，直接从父目录中删除该节点即可，等价于递归删除整棵子树

- `list_dir()`
  - 先返回排序后的目录名
  - 再返回排序后的文件名

### 3. 复杂度

- `make_dir` / `create_file` / `remove`：`O(1)`
- `change_dir`：`O(k)`，`k` 为路径段数
- `list_dir`：`O(d log d + f log f)`，`d` 为目录数，`f` 为文件数

## 修正后的测试样例

题面样例使用的是驼峰命名：

- `makeDir`
- `createFile`
- `changeDir`
- `listDir`

根据当前代码框架，需要改成蛇形命名：

- `make_dir`
- `create_file`
- `change_dir`
- `list_dir`

### 样例 1

```python
fs = FileSystem()
print("null")
print(fs.make_dir("dirc"))
print(fs.make_dir("dirb"))
print(fs.make_dir("dirc"))
print(fs.list_dir())
print(fs.change_dir("dirc/"))
print(fs.create_file("fileb"))
print(fs.make_dir("dirb"))
print(fs.create_file("dirb"))
print(fs.list_dir())
print(fs.change_dir("/dirb/dirc"))
```

输出：

```python
null
True
True
False
['dirb', 'dirc']
True
True
True
False
['dirb', 'fileb']
False
```

### 样例 2

```python
fs = FileSystem()
print("null")
print(fs.list_dir())
print(fs.change_dir(""))
print(fs.create_file("gateway"))
print(fs.make_dir("home"))
print(fs.change_dir("gateway"))
print(fs.make_dir("etc"))
print(fs.list_dir())
print(fs.remove("gateway"))
print(fs.change_dir("etc/"))
print(fs.create_file("pip"))
print(fs.change_dir("/etc/"))
print(fs.list_dir())
print(fs.change_dir("/"))
print(fs.remove("etc"))
print(fs.list_dir())
```

输出：

```python
null
[]
True
True
True
False
True
['etc', 'home', 'gateway']
True
True
True
True
['pip']
True
True
['home']
```
