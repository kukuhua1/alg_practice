# 最小路由时延和（软件认证题）

## 题目描述

某网络中有 `routerNum` 台路由器，编号为 `1 ~ routerNum`，即 `R1 ~ RrouterNum`。

已知某两个路由器之间最多只有一条网线双向连通，每条网线都有一个通信时延 `delay`。由于是双向连通，因此：

- 从 `Ra` 到 `Rb` 的时延
- 与从 `Rb` 到 `Ra` 的时延

二者相等。

现给定一个数组 `linkRelations` 表示路由器之间的连通关系和通信时延，以及指定的两个路由器 `routerRi` 和 `routerRj`，请计算它们之间的最小时延和。

如果两个路由器之间无法连通，返回 `-1`。

## 输入说明

- 第一个参数：`routerNum`，表示路由器个数
  - `1 <= routerNum <= 10000`
- 第二个参数：`linkRelations`，表示连通关系
  - `1 <= linkRelations.length <= 10000`
  - `linkRelations[i] = [routerA, routerB, delay]`
  - `1 <= routerA, routerB <= routerNum`
  - `1 <= delay <= 10000`
- 第三个参数：`routerRi`，表示起始路由器
  - `1 <= routerRi <= routerNum`
- 第四个参数：`routerRj`，表示目标路由器
  - `1 <= routerRj <= routerNum`

## 输出说明

输出一个整数，表示 `routerRi` 和 `routerRj` 之间的最小时延；如果两者未连通，则返回 `-1`。

## 样例

**输入：**

```python
routerNum = 6
linkRelations = [[1, 2, 2], [1, 3, 4], [2, 5, 3], [5, 6, 2]]
routerRi = 2
routerRj = 6
```

**输出：**

```python
5
```

**说明：**

一共 `6` 台路由器，`2` 号路由器到 `6` 号路由器的最短路径为：

```python
2 -> 5 -> 6
```

总时延为：

```python
3 + 2 = 5
```

题目原解释中若写成其他加法，是笔误，正确结果应为 `5`。

## 解题思路

这是一个典型的 **带权无向图最短路径** 问题：

- 路由器是图中的点
- 网线是图中的边
- `delay` 是边权

要求两个点之间的最短路径长度，并且所有边权都为正数，因此适合使用 **Dijkstra 算法**。

## 为什么不用 BFS

BFS 只适用于每条边权重都相同的情况。

而本题中每条网线的时延 `delay` 可能不同，例如有的边权是 `2`，有的边权是 `100`，所以不能用普通 BFS，必须使用 Dijkstra。

## 算法步骤

1. 使用邻接表建图
2. 由于网线双向连通，所以每条边都要加入两次
3. 使用最小堆维护当前“最小时延”的候选节点
4. 每次弹出当前时延最小的路由器进行扩展
5. 如果弹出的节点就是目标路由器，直接返回当前时延
6. 如果最终无法访问到目标路由器，返回 `-1`

## 复杂度分析

设：

- 路由器数量为 `n`
- 连线数量为 `m`

则：

- 时间复杂度：`O((n + m) log n)`
- 空间复杂度：`O(n + m)`

在题目给定的数据范围内，这个复杂度是可以接受的。

## Python 参考实现

```python
import heapq
from collections import defaultdict


def min_router_delay(router_num, link_relations, router_ri, router_rj):
    if router_ri == router_rj:
        return 0

    graph = defaultdict(list)
    for a, b, delay in link_relations:
        graph[a].append((b, delay))
        graph[b].append((a, delay))

    dist = [float("inf")] * (router_num + 1)
    dist[router_ri] = 0
    heap = [(0, router_ri)]

    while heap:
        cur_delay, node = heapq.heappop(heap)

        if cur_delay > dist[node]:
            continue

        if node == router_rj:
            return cur_delay

        for next_node, edge_delay in graph[node]:
            new_delay = cur_delay + edge_delay
            if new_delay < dist[next_node]:
                dist[next_node] = new_delay
                heapq.heappush(heap, (new_delay, next_node))

    return -1
```

## 示例验证

```python
router_num = 6
link_relations = [[1, 2, 2], [1, 3, 4], [2, 5, 3], [5, 6, 2]]
router_ri = 2
router_rj = 6

print(min_router_delay(router_num, link_relations, router_ri, router_rj))
```

输出：

```python
5
```

## 边界情况

- 如果 `routerRi == routerRj`，最小时延为 `0`
- 如果图中没有从 `routerRi` 到 `routerRj` 的路径，返回 `-1`
- 如果某个路由器没有任何连线，也不会影响算法正确性

## 总结

这道题本质是带正权无向图单源最短路问题。

核心做法：

- 邻接表建图
- 使用最小堆优化 Dijkstra
- 找到目标节点时即可提前返回

这是这类“最短时延”“最短费用”“最短距离”题目的标准解法。
