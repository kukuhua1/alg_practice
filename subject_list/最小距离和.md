# 最小距离和（软件认证题）

## 题目描述

终端公司的零售店需要定期去仓库提取货物。零售店和仓库分布在一个二维矩阵 `matrix` 上：

- 相邻位置之间的距离为 `1`
- 只能沿上下左右四个方向移动

矩阵中的元素只可能是以下三种值：

- `0`：表示仓库
- `1`：表示零售店
- `-1`：表示障碍

其中障碍不可通过，其余位置可以通过。

现在需要计算每个零售店到最近仓库的最小距离，并输出这些最小距离之和，规则如下：

- 无法到达任意仓库的零售店，不参与距离和计算
- 如果没有零售店或者没有仓库，返回 `0`

## 输入说明

输入为一个二维数组 `matrix`，满足：

- `1 <= matrix.length, matrix[i].length < 300`
- `matrix[i][j]` 的取值只可能为 `0`、`1`、`-1`

## 输出说明

输出一个整数，表示所有可到达仓库的零售店到最近仓库的最小距离之和。

## 样例

**输入：**

```python
[
    [1, -1, 0],
    [0, 1, 1],
    [1, -1, 1]
]
```

**输出：**

```python
6
```

**说明：**

矩阵中共有 `5` 个零售店、`2` 个仓库、`2` 个障碍。

- `matrix[2][2]` 到最近仓库的最小距离为 `2`
- 其余 `4` 个零售店到最近仓库的最小距离均为 `1`

因此结果为：

```python
1 + 1 + 1 + 1 + 2 = 6
```

## 解题思路

这道题的关键在于：

- 每个零售店都要找到“最近仓库”
- 地图上的每条可行边权重都相同，都是 `1`

这种场景非常适合使用 **多源 BFS**。

### 为什么要用多源 BFS

如果从每个零售店分别做一次 BFS，最坏情况下会非常慢。

更高效的做法是：

1. 把所有仓库位置同时加入队列，作为 BFS 的起点
2. 从这些仓库同时向外扩散
3. 每个可达位置第一次被访问到时，得到的就是它到最近仓库的最短距离

这样只需要一次 BFS，就能求出所有位置到最近仓库的距离。

## 算法步骤

1. 遍历矩阵，统计零售店和仓库的位置
2. 如果没有零售店或没有仓库，直接返回 `0`
3. 将所有仓库加入 BFS 队列，初始距离设为 `0`
4. 进行多源 BFS，跳过障碍和越界位置
5. BFS 结束后，再遍历一次矩阵
6. 对于值为 `1` 且距离不为 `-1` 的位置，将距离累加到答案中

## 复杂度分析

设矩阵大小为 `m x n`。

- 时间复杂度：`O(m * n)`
- 空间复杂度：`O(m * n)`

每个位置最多只会入队一次，因此整体效率较高。

## Python 参考实现

```python
from collections import deque


def min_distance_sum(matrix):
    if not matrix or not matrix[0]:
        return 0

    rows, cols = len(matrix), len(matrix[0])
    dist = [[-1] * cols for _ in range(rows)]
    queue = deque()

    store_count = 0
    warehouse_count = 0

    for i in range(rows):
        for j in range(cols):
            if matrix[i][j] == 1:
                store_count += 1
            elif matrix[i][j] == 0:
                warehouse_count += 1
                dist[i][j] = 0
                queue.append((i, j))

    if store_count == 0 or warehouse_count == 0:
        return 0

    directions = [(1, 0), (-1, 0), (0, 1), (0, -1)]

    while queue:
        x, y = queue.popleft()

        for dx, dy in directions:
            nx, ny = x + dx, y + dy

            if not (0 <= nx < rows and 0 <= ny < cols):
                continue
            if matrix[nx][ny] == -1:
                continue
            if dist[nx][ny] != -1:
                continue

            dist[nx][ny] = dist[x][y] + 1
            queue.append((nx, ny))

    ans = 0
    for i in range(rows):
        for j in range(cols):
            if matrix[i][j] == 1 and dist[i][j] != -1:
                ans += dist[i][j]

    return ans
```

## 示例验证

对于样例：

```python
matrix = [
    [1, -1, 0],
    [0, 1, 1],
    [1, -1, 1]
]

print(min_distance_sum(matrix))
```

输出：

```python
6
```

## 总结

这道题本质上是一个二维网格最短路问题，但由于所有边权都相同，所以不需要 Dijkstra，直接使用多源 BFS 即可。

核心结论：

- 从所有仓库同时出发
- 一次 BFS 求出所有点到最近仓库的距离
- 最后累加所有可达零售店的最短距离
